# 🔐 Sistema de Login

Projeto simples de **autenticação de usuários** desenvolvido com **Python, Flask e SQLite**.
O sistema permite cadastrar usuários, fazer login, visualizar usuários cadastrados e deletar usuários no dashboard.

---

# 📌 Funcionalidades

✔ Cadastro de usuários
✔ Login de usuários
✔ Sessão de usuário com Flask
✔ Dashboard com lista de usuários cadastrados
✔ Contagem total de usuários
✔ Exclusão de usuários
✔ Validação básica de email
✔ Interface simples com HTML e CSS

# ▶️ Como executar o projeto

 1️⃣ Clonar o repositório

```
git clone https://github.com/seuusuario/seu-repositorio.git
```

---

 2️⃣ Entrar na pasta do projeto

```
cd seu-repositorio
```

---

 3️⃣ Instalar as dependências

```
pip install flask
```

---

 4️⃣ Executar o projeto

```
python app.py
```

---

5️⃣ Abrir no navegador

```
http://127.0.0.1:5000
```



📷 Demonstração

O sistema possui três telas principais:

🔑 Login

Tela para autenticação do usuário.

📝 Cadastro

Tela para criação de novos usuários.

📊 Dashboard

Painel que mostra:

* usuário logado
* lista de usuários cadastrados
* total de usuários
* opção de deletar usuários

---
🧠 Aprendizados com este projeto

Durante o desenvolvimento foram praticados conceitos como:

* criação de rotas no Flask
* manipulação de banco de dados SQLite
* criação de templates com Jinja
* gerenciamento de sessão de usuários
* integração entre backend e frontend
* validação de dados
